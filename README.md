Esta é uma Landing Page de alta performance desenvolvida para o consultor financeiro **Wagner Teles Alvaro**. O projeto une um design moderno em **Bootstrap 5** com técnicas avançadas de **Copywriting (Método AIDA)** para maximizar a geração de leads e fechamentos de empréstimos via WhatsApp.

---

## 🎯 Objetivo do Projeto

Posicionar a marca **Poder Financeiro** como autoridade no setor de crédito, oferecendo uma solução rápida e segura para brasileiros que buscam "respiro financeiro". A página é otimizada para converter visitantes em clientes reais através de gatilhos de urgência e prova de transparência.

## 🛠️ Stack Tecnológica

- **Frontend:** HTML5, CSS3 (Customizado) e Bootstrap 5.3.
- **Ícones:** Bootstrap Icons.
- **Copywriting:** Framework AIDA (Atenção, Interesse, Desejo, Ação) e Gatilhos Mentais.
- **Integração:** Direct Link WhatsApp API.

## ✨ Diferenciais Técnicos e de Negócio

* **Headline 4 U's:** Foco em Urgência, Utilidade, Ultra-especificidade e Unicidade (Baseada na Resolução CCFGTS nº 1.130/2025).
* **Design Responsivo (Mobile-First):** Experiência fluida em smartphones, onde ocorre 90% dos acessos para este nicho.
* **Animações de Conversão:** Botão de WhatsApp flutuante e CTAs com efeito *Pulse* para guiar o olho do usuário.
* **Seção de Transparência:** Bloco dedicado às "Regras de Ouro", combatendo objeções e aumentando a confiança.
* **Segurança:** Rodapé com aviso anti-golpe, reforçando a integridade do consultor.
